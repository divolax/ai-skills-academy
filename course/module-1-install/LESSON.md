# Модуль 1: Установка + Первый Вау-Момент

> Время: ~10 минут

---

## Что ты сделаешь

1. Установишь Claude Code
2. Запустишь его первый раз
3. AI узнает тебя и твои задачи
4. Увидишь свой первый "вау"-момент

---

## Шаг 1: Установка Claude Code

Открой терминал:
- **Mac:** Нажми Cmd+Space, введи "Terminal", нажми Enter
- **Windows:** Нажми Win+R, введи "powershell", нажми Enter

Вставь команду для своей системы:

**Mac / Linux:**
`ash
curl -fsSL https://claude.ai/install.sh | bash
`

**Windows (PowerShell):**
`powershell
irm https://claude.ai/install.ps1 | iex
`

Дождись окончания установки. Это займёт 1-2 минуты.

---

## Шаг 2: Первый запуск

Введи в терминале:
`ash
claude
`

Ты увидишь приветствие Claude. Поздоровайся с ним!

---

## Шаг 3: AI узнает тебя

Скажи Claude следующее (скопируй и вставь):

`
Привет! Я хочу, чтобы ты стал моим AI-ассистентом для бизнеса и жизни. 
Задай мне 5 вопросов, чтобы понять:
1. Чем я занимаюсь
2. Какие у меня цели
3. Какие задачи я делаю каждый день
4. Что меня больше всего раздражает в рутине
5. Чего я хочу достичь с AI
`

Claude задаст тебе вопросы. Отвечай честно — чем больше он знает, тем полезнее будет.

---

## Шаг 4: Создаём твой CLAUDE.md

Теперь скажи Claude:

`
Создай файл CLAUDE.md на основе наших ответов. 
Включи туда: мою нишу, аудиторию, стиль общения, инструменты, цели.
`

Этот файл — твой "мозг" для AI. Каждый раз, когда ты открываешь Claude, он будет читать этот файл и знать, кто ты.

---

## Шаг 5: Первый вау-момент

Теперь самое интересное. Скажи Claude:

`
Используя мой CLAUDE.md, создай мне контент-план на неделю для моего бизнеса.
`

Смотри, как AI за 30 секунд делает то, на что у тебя ушёл бы час.

**Это только начало. Дальше — больше.**
