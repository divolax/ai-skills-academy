# Модуль 4: Бот-Билдер

> Время: ~15 минут

---

## Что ты сделаешь

1. Создашь своего первого Telegram-бота
2. Бот будет собирать заявки
3. Научишься добавлять функции

---

## Шаг 1: Создай бота в BotFather

1. Открой Telegram
2. Найди [@BotFather](https://t.me/BotFather)
3. Напиши /newbot
4. Следуй инструкциям
5. Скопируй токен (выглядит как: 123456789:ABCdef...)

---

## Шаг 2: Открой skill бот-билдера

В Claude напиши:
`
/ai:botbuilder
`

Или скажи:
`
Я хочу создать Telegram-бота для сбора заявок
`

---

## Шаг 3: Опиши, что должен делать бот

Расскажи Claude:
- Для чего бот (заявки, поддержка, продажи)
- Какие данные собирать (имя, телефон, email)
- Куда отправлять уведомления (тебе в личку, в группу)

---

## Шаг 4: Claude пишет код

Claude создаст:
- Полный код бота на Python
- Инструкцию по запуску
- Файл requirements.txt
- README для бота

---

## Шаг 5: Запусти бота

`ash
# Установи библиотеку
pip install python-telegram-bot

# Запусти бота
python bot.py
`

Твой бот работает! Протестируй его в Telegram.
