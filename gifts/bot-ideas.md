# 10 Идей Ботов для Бизнеса

Готовые идеи, которые можно реализовать с помощью /ai:botbuilder

---

## 1. Бот для сбора заявок
Что делает:
- Принимает имя, телефон, email
- Отправляет заявку тебе в личку
- Сохраняет в таблицу

Для кого: Любой бизнес, который принимает заявки

---

## 2. Бот-визитка
Что делает:
- Показывает информацию о компании
- Каталог услуг/товаров
- Контакты и адрес

Для кого: Малый бизнес, фрилансеры

---

## 3. Бот для поддержки
Что делает:
- Отвечает на частые вопросы (FAQ)
- Передаёт сложные вопросы человеку
- Собирает обратную связь

Для кого: Компании с большим количеством обращений

---

## 4. Бот для продаж
Что делает:
- Показывает товары с фото и ценами
- Принимает заказы
- Отправляет реквизиты для оплаты

Для кого: Интернет-магазины, продавцы

---

## 5. Бот для записи
Что делает:
- Показывает свободные слоты
- Записывает на приём
- Напоминает о записи

Для кого: Салоны красоты, врачи, консультанты

---

## 6. Бот для обучения
Что делает:
- Отправляет уроки по расписанию
- Принимает домашние задания
- Даёт обратную связь

Для кого: Онлайн-школы, коучи

---

## 7. Бот для опросов
Что делает:
- Задаёт вопросы
- Собирает ответы
- Формирует отчёт

Для кого: Маркетологи, исследователи

---

## 8. Бот для контент-рассылки
Что делает:
- Рассылает посты по расписанию
- Сегментирует аудиторию
- Собирает статистику

Для кого: Блогеры, медиа

---

## 9. Бот для трекинга привычек
Что делает:
- Напоминает о привычках
- Считает streak
- Показывает статистику

Для кого: Личное использование, коучи

---

## 10. Бот для анализа питания
Что делает:
- Принимает фото еды
- Считает КБЖУ
- Ведёт дневник питания
- Показывает статистику

Для кого: Фитнес-тренеры, нутрициологи, личное использование
